##
# Twitter to mixiボイス accounts
# 
# タブまたはスペース区切りで
# 　mixiログインメールアドレス mixiパスワード twitterユーザ名 twitterパスワード   投稿ハッシュタグ(オプション)
# の順に記入してください。
# 
# 投稿ハッシュタグ
#   #hash  はそのハッシュタグが付与されているメッセージのみ投稿されます。(肯定フィルター)
#   !#hash はそのハッシュタグが付与されているメッセージは投稿されません。(否定フィルター)
# 複数の指定がある場合は、まず否定フィルターが評価され、その後肯定フィルターが評価されます。
# それぞれの評価はOR条件として結合されます。
# 
# 例)
# ハッシュタグ指定なし 全てのメッセージがmixiへ投稿されます。
# #a               "#a"を含む投稿のみmixiへ投稿されます。
# !#a              "#a"を含む投稿はmixiへ投稿されません。
# #a #b            "#a"または"#b"を含む投稿のみmixiへ投稿されます。
# !#a !#b          "#a"または"#b"を含む投稿はmixiへ投稿されません。
# !#a !#b #c #d    "#a"または"#b"を含まず、"#c"または"#d"を含む投稿のみmixiへ投稿されます。
#
