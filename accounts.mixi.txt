##
# mixiボイス to Twitter accounts
# 
# タブまたはスペース区切りで
# 　mixiログインメールアドレス mixiパスワード twitterユーザ名 twitterパスワード   投稿キーワード(オプション)
# の順に記入してください。
# 
# 投稿キーワード
#   keyword  はそのキーワードが付与されているメッセージのみ投稿されます。(肯定フィルター)
#   !keyword はそのキーワードが付与されているメッセージは投稿されません。(否定フィルター)
# 複数の指定がある場合は、まず否定フィルターが評価され、その後肯定フィルターが評価されます。
# それぞれの評価はOR条件として結合されます。
# 
# 例)
# キーワード指定なし 全てのメッセージがtwitterへ投稿されます。
# a              "a"を含む投稿のみtwitterへ投稿されます。
# !a             "a"を含む投稿はtwitterへ投稿されません。
# a b            "a"または"b"を含む投稿のみtwitterへ投稿されます。
# !a !b          "a"または"b"を含む投稿はtwitterへ投稿されません。
# !a !b c d      "a"または"b"を含まず、"c"または"d"を含む投稿のみtwitterへ投稿されます。
#
# [注意]
# このファイルをWeb公開ディレクトリ等、他者から閲覧可能な場所に設置しないでください。
